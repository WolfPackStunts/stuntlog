1
Set Up Your Profile
Go to ⚙️ Settings — enter your name and SAGA/union number.
2
Create a Project
Go to 🎬 Projects — add the show title, company, start date, person in charge, and your default day rate.
3
Log Your Days
Go to 📋 Log Day — select project, day type, call/wrap times. OT auto-calculates. Add travel, stunt adj, hire, SASA.
4
Review & Edit
Go to 📁 Entries — all days grouped by shoot week. Edit or delete any entry.
5
Generate Timesheets
Go to 📄 Timesheet — select project and week, generate and copy.
• OT auto-calculates from call & wrap times
• Set day rate in Projects — auto-fills each day
• Turnaround auto-checks from previous entry
• Night shoots? Use the Overnight toggle
• Data saves automatically on your device
